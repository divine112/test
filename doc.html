How To Install Script

1.Upload and unzip vmark.zip in to your webserver
2. Chnage The config.php file according to your needs.
↳ $secretPassword = "YOUR MD5 HSAHSED PASSWORD"; Default Password is demo
↳ Set $ffmpeg if your ffmpeg location is diffrent
↳ Set $thumbs_size For Cusom Thumbnail Size Image (Use 150:150 for 150px by 150px
↳ Set $keepfiles true or false. If its True it will keep originalfiles and create new file for watermarked.
3. Add your watermark logo files into watermakrs folder with .png extension
4. For Video Watermark. Add .mov file with alpha and upload png file of same video size with correct file names.
i.e. if file name is wm.mov image file name is wm.mov.png
and upload both on watermark_video folder
5. For Audio Replace , Upload your audio/mp3 on audio folder.
You are ready To Use This Script Click here Start V Mark

Video Watermark: NEW in V4

1.In List Page Click On Button LIKE : VMark It! > Choose Plugin IMAGE-FADE
2. Choose VIDEO Option.
3. Choose Video Watermark From The Drop Down.
4. Once Selected You Will See An Image, Click on image and you able to drag and resize image from bottom right.
5. Place to correct position and set time for the video overlay. You can also set multiple times on same video.
(This will useful if you wanted to show transparent overlay video, lowerthierd , animated watermark etc. )
6. Once Time Is Set. Double click on watermark image to get the posistion and scale.
7. Click On Render Video Button Will Start The Process.
